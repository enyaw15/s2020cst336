CSS

CSS or Cascading Style Sheets is used to add formating, colors, and other styles to html documents. CSS can be put directly into HTML documents or it can be put into its own css document. CSS makes websites look good and with the help of JavaScript css and react to user inputs to create dynamic styles. CSS is a useful language making websites look great.
